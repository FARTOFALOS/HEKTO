# 🔰 УСТАНОВКА HEKTO — Пошаговая инструкция для новичков

**Ты не программист? Не знаешь что такое терминал? Не беда — следуй шагам ниже.**

Всего 3 шага: **Скачать → Открыть терминал → Запустить одну команду**.

---

## Шаг 0: Что тебе понадобится

- 💻 Ноутбук (Windows, Mac или Linux)
- 🌐 Интернет
- 🎙 Микрофон (встроенный в ноутбук подойдёт)
- ⏱ ~10 минут на установку

---

## Шаг 1: Скачай проект

### Вариант А — Скачать ZIP (самый простой)

1. Открой в браузере: **https://github.com/FARTOFALOS/HEKTO**
2. Нажми зелёную кнопку **`<> Code`** (она в правом верхнем углу списка файлов)
3. В выпадающем меню нажми **`Download ZIP`**
4. Найди скачанный файл `HEKTO-main.zip` (обычно в папке «Загрузки»)
5. **Распакуй** его (правый клик → «Извлечь всё» / «Extract All»)
6. Запомни куда распаковал — например `C:\Users\Твоё_Имя\Downloads\HEKTO-main`

### Вариант Б — Через Git (если знаешь что это)

```bash
git clone https://github.com/FARTOFALOS/HEKTO.git
cd HEKTO
```

---

## Шаг 2: Открой терминал в папке проекта

### 🪟 Windows

**Способ 1 — Самый простой:**
1. Открой распакованную папку `HEKTO-main` в Проводнике
2. Кликни на адресную строку (где написан путь к папке вверху)
3. Напиши `cmd` и нажми **Enter**
4. Откроется чёрное окно терминала — ты уже в нужной папке!

**Способ 2 — Через меню:**
1. Нажми **Win + R** (кнопка Windows + R)
2. Напиши `cmd` и нажми Enter
3. В чёрном окне напиши:
   ```
   cd C:\Users\Твоё_Имя\Downloads\HEKTO-main
   ```
   (замени путь на свой)

### 🍎 Mac

1. Открой **Finder** → найди распакованную папку `HEKTO-main`
2. Кликни правой кнопкой на папку
3. Выбери **«Службы» → «Новый терминал в папке»**

   Или:
1. Открой **Terminal** (найди через Spotlight: Cmd+Пробел → набери "Terminal")
2. Напиши `cd ` (с пробелом!) и **перетащи папку** из Finder в окно Terminal
3. Нажми Enter

### 🐧 Linux

1. Открой файловый менеджер → найди папку `HEKTO-main`
2. Правый клик → **«Открыть терминал здесь»**

## Шаг 3: Запусти установку

### 🪟 Windows

В терминале (cmd) набери:

```
setup_hekto.bat
```

Или просто **дважды кликни** на файл `setup_hekto.bat` в папке проекта.

В терминале набери:


## 🚨 Если Python не установлен

Скрипт сам скажет, что его нет. Вот как установить:

### Windows
1. Открой https://www.python.org/downloads/
2. Нажми **"Download Python 3.12"** (или новее)
3. Запусти скачанный файл
4. ⚠️ **ОБЯЗАТЕЛЬНО** поставь галочку **"Add Python to PATH"** на первом экране!
5. Нажми "Install Now"
6. После установки **закрой и снова открой** терминал
7. Снова запусти `setup_hekto.bat`

### Mac
```bash
# Сначала установи Homebrew (если ещё нет):
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Потом Python:
brew install python@3.12
```

### Linux (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install -y python3 python3-venv python3-pip
```

## ❓ Частые проблемы

### «Python не найден» / «python is not recognized»
→ Установи Python (см. выше) и **не забудь галочку "Add to PATH"**

### «FFmpeg not found»
→ На Windows: скрипт подскажет. На Mac: `brew install ffmpeg`

### «No microphone found»
→ Это нормально при установке. Главное чтобы микрофон работал когда записываешь.

### «Permission denied» (Mac/Linux)
→ Попробуй: `chmod +x setup_hekto.sh && bash setup_hekto.sh`

### Хочу проверить, всё ли работает
```bash
python setup_hekto.py --check
```
